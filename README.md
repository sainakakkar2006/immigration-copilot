# Immigration Co-Pilot

**Your immigration questions deserve real answers.** A free web tool for immigrant families navigating the US immigration system — built for people who can't afford an attorney and need plain-language guidance on what's happening and what to do next, in their own language.

**Live app:** https://immigration-copilot-adazqhaecrqc8g7dujpe7h.streamlit.app/

Built for the **Build with Gemini XPRIZE** (XPRIZE Foundation + Google).

## Why this matters

- Only ~30% of immigrants with pending court cases have a lawyer ([TRAC](https://tracreports.org/reports/736/))
- 62% of people without a lawyer are ordered deported, vs 27% with one ([American Immigration Council](https://www.americanimmigrationcouncil.org/report/access-counsel-immigration-court/))
- Nearly all AI immigration tooling is sold to attorneys. This tool is for families.

## What it does

| Feature | How it works |
|---|---|
| **Decode a Letter** | Photograph any USCIS/ICE/State Dept notice (RFE, NOID, receipt, denial). Gemini reads it natively — even phone photos — extracts the notice type, case number, and every deadline, explains it at a 5th-grade level, and gives you a calendar (.ics) file with reminders. |
| **News** | Real headlines from USCIS, the White House, the State Department, and the Federal Register, fetched by a GitHub Action every 2 days and explained in plain language by Gemini. When feeds are quiet, Gemini searches the live web (Google Search grounding) and cites what it finds. |
| **Your Situation** | Describe your case, optionally attach a document (PDF or photo), and get an honest assessment: issues, urgency, actions, deadlines, and official resources. |
| **OPT Deadlines** | Every OPT / STEM OPT deadline computed deterministically in code from USCIS rules — the AI explains dates, it never invents them. Includes calendar download. |
| **Green Card Checklist** | Case-specific forms, fees, document checklist, red flags, timeline, and a ready-to-edit cover letter. Downloadable to share with family or an attorney. |
| **12 languages** | The entire interface and every answer available in English, Spanish, Chinese, Hindi, Tagalog, Vietnamese, Korean, Portuguese, Arabic, Haitian Creole, Russian, and French — translated by Gemini and cached. |

## How Gemini is used

- **Multimodal document understanding** — PDFs and phone photos of government letters read natively (`Part.from_bytes`), no OCR pipeline
- **Google Search grounding** — live, cited policy news instead of training-data guesses
- **Structured JSON output** — every feature gets typed, render-ready responses
- **On-the-fly interface translation** — one cached Gemini call per language

Model: `gemini-2.5-flash` via the `google-genai` SDK.

## Architecture

```
GitHub Action (every 2 days)          Streamlit Community Cloud
┌───────────────────────────┐         ┌────────────────────────────┐
│ fetch_news.py             │ commits │ app.py                     │
│ USCIS / WH / State RSS    │────────▶│ reads data/uscis_news.json │
│ + Federal Register API    │         │ + Gemini for everything AI │
└───────────────────────────┘         └────────────────────────────┘
```

Government sites block cloud-provider IPs, so RSS is fetched on GitHub's runners and committed to the repo; the app reads the JSON locally. The Federal Register API allows direct calls and is cached for 48 hours.

## Run it locally

```bash
pip install -r requirements.txt
export GEMINI_API_KEY=your-key   # from aistudio.google.com
streamlit run app.py
```

## Privacy & safety

- Uploaded documents are processed in memory and never stored
- No accounts, no tracking, no immigration-status data retained
- Every page carries a "not legal advice" notice with a one-tap link to free legal aid ([immigrationlawhelp.org](https://www.immigrationlawhelp.org))
- Deadline math is computed in code, never generated by AI

## Disclaimer

This is not a law firm and does not give legal advice. If your case involves prior violations, criminal history, or deportation proceedings, consult a licensed immigration attorney.
